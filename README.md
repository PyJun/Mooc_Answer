## 学无止答题器

### 1.软件介绍：

- 该软件用于中国大学慕课自动答题，包括自动浏览视频，自动提交主观题和客观题答案，以及参与讨论。
- 软件采用Vip收费机制，普通用户可免费答题三次，Vip用户可以答题无限次，Vip有包日，包月，包年。
- Vip充值采用激活码的形式充值，激活码购买链接：[点此购买](https://pr.kuaifaka.com/item/vv7Y51)
